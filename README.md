# Meus Drinks

App mobile em **Ionic + Angular** que consome a API publica [TheCocktailDB](https://www.thecocktaildb.com/) para buscar drinks e salvar favoritos.

Projeto desenvolvido para a **Avaliacao Pratica de Laboratorio** (Formadora IV).

## Tecnologias

- Ionic 8
- Angular 20
- TheCocktailDB API

## Funcionalidades

- Buscar drinks por nome
- Sortear um drink aleatorio
- Ver detalhes do drink (ingredientes e modo de preparo)
- Favoritar drinks com avaliacao e anotacao
- Listar, editar e excluir favoritos

## Como rodar

```bash
npm install
ionic serve
```

Acesse: [http://localhost:8100](http://localhost:8100)
